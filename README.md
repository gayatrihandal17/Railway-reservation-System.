# Railway Reservation System 🚆

## 📌 Project Description

This project is a Railway Reservation System developed using Java. It allows users to book train tickets, view available trains, and manage reservations efficiently.

## 🚀 Features

* User registration and login
* Search trains
* Book tickets
* Cancel tickets
* View booking details

## 🛠️ Technologies Used

* Java
* Maven
* Eclipse IDE
* MySQL (if used)

## ▶️ How to Run

1. Clone the repository
2. Open project in Eclipse
3. Run the main file
4. Follow instructions on console

## 📂 Project Structure

* src/ → Source code
* pom.xml → Maven configuration
